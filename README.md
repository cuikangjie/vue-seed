# Vue-seed

> by kin

## 项目介绍

> 项目为整合 vue+vuex+vue-router+elementUi 克隆后可快速开发vue项目

> 实现了vuejs的filter、mixin

> 实现了vuex开发的基本流程

> 实现了vue-router的一级路由配置

>  实现了elementUi(1.3.0) 安需导入组件 [自定义主题(#657893)](http://element.eleme.io/#/zh-CN/component/custom-theme)

## 项目启动

```bash
# 安装依赖
npm install

# 启动服务  ip:8888/dist/
npm run dev

# 打包
npm run build
```
